# 📱 UIElements

Bienvenido a **UIElements**, una aplicación móvil nativa desarrollada en Kotlin con Android Studio.  
Este proyecto sirve como ejemplo de uso de diferentes componentes de UI modernos y navegación entre Activities y Fragments en Android.

---

## 🚀 ¿Qué hace esta app?

- Demuestra la implementación de distintos elementos de interfaz (botones, listas, campos de texto, selección, etc).
- Permite la navegación entre varios fragments mediante un menú inferior.
- Incluye ejemplos de comunicación entre Activities (por ejemplo, abrir una segunda Activity y pasar datos).
- Incluye el uso de Material Design y estilos modernos.

---

## 🧑‍💻 Estructura

- **MainActivity.kt:** Activity principal que contiene el menú inferior y gestiona los fragments.
- **Fragments:** Secciones independientes de la UI: texto, botones, selección, listas, info.
- **SecondActivity.kt:** Otra Activity de ejemplo que recibe datos y permite regresar o cerrar.
- **Layouts:** XMLs organizados en la carpeta `res/layout` para cada pantalla y fragmento.

## 📦 Cómo ejecutar

1. Clona el repositorio:
   ```sh
   git clone https://github.com/gael-marquez/UIElements.git
   ```
2. Ábrelo en Android Studio.
3. Ejecuta el proyecto en un emulador o dispositivo físico.

---

## 🎯 Objetivo educativo

Este proyecto está pensado para estudiantes, desarrolladores junior o cualquier persona que quiera aprender:
- Cómo estructurar una app con múltiples fragments.
- Cómo navegar entre Activities y pasar datos.
- Cómo aplicar Material Design en Android.
